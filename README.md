[v1.0.2] - 2026-03-07
新增功能 (Features)
右侧详情栏 (Right Sidebar) 实现
新增 RightAside 组件，采用响应式布局（md 断点显隐），支持折叠/展开交互（宽度 384px ⇄ 120px 平滑过渡）。
当前播放视图：集成大幅方形专辑封面、高亮歌曲标题及艺人名称显示。
快捷收藏：在当前播放信息区集成“红心”收藏按钮，支持实时鉴权、状态切换及 ElMessage 反馈。
艺人百科卡片：
实现了基于当前播放曲目的艺人信息动态获取机制（监听 currentTrack.artist）。
集成 getAllArtists 与 getArtistDetail API，自动拉取艺人头像及简介。
支持点击艺人卡片跳转至艺人详情页 (/artist/:id)。
播放队列预览：
新增“下一首”卡片，实时计算并展示队列候播曲目信息。
支持点击卡片直接切歌 (playNext)。
界面与交互 (UI/UX)
深色模式适配：全组件支持 Dark Mode，背景色采用层次化灰度设计 (#121212 / #1e1e1e)，文本颜色自动反转。
视觉优化：
引入 Spotify 风格的卡片式排版。
图片资源增加加载兜底（Default Cover/Avatar）。
文本区域（如艺人简介）增加行数限制与溢出省略处理。
滚动体验：定制化 CSS 滚动条 (custom-scrollbar)，隐藏原生滚动条并优化滑块视觉。
交互微效：添加封面悬停播放遮罩、按钮点击波纹反馈及折叠按钮的 Icon 切换动画。
核心逻辑 (Core Logic)
数据清洗：针对后端返回的富文本艺人简介，增加了正则过滤器 (replace(/<[^>]+>/g, '')) 以移除 HTML 标签，确保纯文本展示安全整洁。
状态管理：重构了播放状态监听逻辑，利用 watch 深度监听音轨变化，并使用 computed 属性实时计算收藏状态与下一首曲目，减少不必要的重渲染。
异常处理：增强了 API 调用链的健壮性，为艺人信息获取失败、未登录操作添加了完整的 try-catch 捕获与降级处理策略。
